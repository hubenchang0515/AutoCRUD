

# 列的属性及默认值
| 属性名      | 默认值  | 说明 |
| :-         |  :-     | :-   |
| `name`     |         |  列名，必须指定  |
| `type`     |         |  数据类型，必须指定 | 
| `index`    | `false` |  是否创建索引 |
| `nullable` | `true`  |  是否可以位空 | 
| `unique`   | `false` |  是否创建唯一性约束 | 
| `default`  | `null`  |  默认值 | 
| `private`  | `false` |  是否为私有，即返回数据不包含此字段 |
